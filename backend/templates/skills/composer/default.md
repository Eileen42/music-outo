# Default Composer Skill

## 기본 작곡 규칙
- Suno 프롬프트는 반드시 6줄 형식: Genre, Mood, Tempo, Instrumentation, Sound Effects/Atmosphere, Additional Descriptors
- 같은 플레이리스트 내 곡들은 일관된 톤 유지
- BPM과 악기 구성은 프로젝트 전체에서 통일

## Suno 프롬프트 작성 팁
- Genre는 구체적으로 (예: "Ambient Sleep Music" > "Ambient")
- Mood에 감정을 3~5개 키워드로 표현
- Instrumentation에 실제 악기명 사용 (예: "Rhodes piano" > "piano")
- Additional Descriptors에 사운드의 질감 묘사

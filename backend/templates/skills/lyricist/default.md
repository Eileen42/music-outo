# Default Lyricist Skill

## 기본 작사 규칙
- Suno AI가 인식하는 구조 태그 사용: [Verse], [Chorus], [Bridge], [Outro]
- 한 Verse는 4~8줄
- Chorus는 반복 가능하며 기억에 남는 핵심 메시지
- 가사 언어는 영어 기본 (사용자 요청 시 변경 가능)

## Suno 가사 태그 가이드
- [Intro] — 도입부 (짧은 허밍이나 1~2줄)
- [Verse 1], [Verse 2] — 이야기 전개
- [Pre-Chorus] — 코러스 진입 빌드업
- [Chorus] — 핵심 메시지 반복
- [Bridge] — 분위기 전환
- [Outro] — 마무리
- [Instrumental] — 가사 없는 연주 구간

## 작사 팁
- 구체적 이미지 사용 (추상적 표현 < 감각적 묘사)
- 운율(rhyme)을 자연스럽게 활용
- 한 곡의 가사가 하나의 주제를 관통하도록

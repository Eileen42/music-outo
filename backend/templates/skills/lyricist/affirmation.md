# 확언/명상 문구 작성 스킬

## 확언(Affirmation) 특성
수면/명상 영상에 자막으로 표시되는 긍정 확언 문구.
시각적으로 읽히며 마음의 안정을 돕는다.

## 작성 규칙
- 한국어, 1문장
- "나는~", "나의~", "오늘~", "이 순간~" 으로 시작
- 현재 시제 사용 (미래형 X)
- 따뜻하고 포근한 어조

## 주제 패턴
- 자기 긍정: "나는 충분히 가치 있는 사람이다"
- 평화/안식: "이 순간, 모든 것이 평화롭다"
- 감사: "오늘 하루도 감사한 마음으로 마무리한다"
- 치유: "나의 몸과 마음이 깊이 회복되고 있다"
- 사랑: "나는 사랑받을 자격이 있는 존재다"

## 피해야 할 것
- 부정적 단어 (못, 안, 없이, 두려움)
- 조건부 표현 ("~하면 좋겠다")
- 명령형 ("~해라")

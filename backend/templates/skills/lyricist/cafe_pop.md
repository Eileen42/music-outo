# 카페/POP 작사 스킬

## 장르별 가사 특성
카페/POP 가사는 일상적이고 따뜻한 이야기를 담는다.
복잡한 메시지보다 감성적 공감을 우선한다.

## 주제 패턴
- 일상의 소소한 행복 (커피 한 잔, 창밖 풍경, 산책)
- 사랑과 그리움 (담담한 톤)
- 계절 변화와 자연
- 도시 속 여유로운 순간

## 가사 스타일
- 대화체 느낌의 친근한 문장
- 감각적 묘사 (향기, 온도, 빛)
- 반복적이고 기억에 남는 코러스
- 2~3절 + 코러스 + 브릿지 구조

## 피해야 할 것
- 과도하게 슬프거나 무거운 주제
- 정치적/사회적 메시지
- 추상적이고 난해한 표현

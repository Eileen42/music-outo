# 카페/POP 음악 설계 스킬

## 곡 구조 패턴
- Intro(8마디) → Verse(16마디) → Chorus(8마디) → Verse2 → Chorus → Outro
- 각 곡: 3~4분
- 후크(Hook)가 명확한 멜로디 라인

## 분위기 변형 테크닉
- **아침 세트** (1~5곡): 밝고 경쾌, Acoustic 위주
- **오후 세트** (6~12곡): 차분하면서 그루비, Lo-fi 요소
- **저녁 세트** (13~17곡): 감성적, Bossa Nova/Jazz 터치
- **마무리** (18~20곡): 따뜻하고 포근

## 서브장르 믹스
하나의 플레이리스트에 여러 서브장르를 자연스럽게 섞기:
- Acoustic Pop → Lo-fi → Bossa Nova → Jazz Pop → Acoustic
- 급격한 전환 없이 점진적으로

## 카테고리별 설계
- morning: 밝고 활기찬, Major key
- relax: 차분하고 여유로운, 7th 코드
- drive: 리듬감 있는, 미드템포

# 수면/명상 음악 설계 스킬

## 곡 구조 패턴
- 각 곡: 3~5분 (짧은 곡이 편집에 유리)
- 곡간 전환: 페이드인/아웃으로 자연스럽게
- 반복: 같은 코드 진행의 미세 변형으로 최면 효과

## 분위기 변형 테크닉
- **Texture 변화**: Piano → Strings → Pads → Piano 순환
- **Sound Layer**: 자연음(비, 바람, 새) 레이어 ON/OFF
- **Key 변화**: C → G → D → A → E (4도 진행으로 점점 밝아지다가 다시 어두워지기)
- **Register**: 점점 낮은 음역대로 이동 (수면 유도)

## 카테고리별 설계
- sleep: 가장 느리고 고요. 변화 최소.
- meditation: 약간의 멜로디. 호흡 리듬 반영.
- healing: 따뜻한 멜로디. 감정적 위로.
- focus: 미세한 리듬감. 반복적 패턴.

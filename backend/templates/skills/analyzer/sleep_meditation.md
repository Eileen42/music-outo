# 수면/명상 음악 분석 스킬

## 타겟 청취자 특성
- 수면 장애, 불면증으로 잠들기 어려운 사람
- 명상/요가 수련자
- 공부/작업 중 집중 BGM 필요한 학생/직장인
- 스트레스 해소가 필요한 직장인

## 성공 채널 패턴
- 영상 길이: 1~10시간 (긴 영상이 조회수 높음)
- 곡 수: 15~30곡이 한 영상에 편집
- 썸네일: 파란색/보라색 톤, 달/별/구름 이미지
- 제목 키워드: "Deep Sleep", "Healing", "Relaxation", "8 Hours"

## 분위기 흐름 최적 패턴
1. 시작(1~3곡): 가볍게 이완 유도 — 아직 깨어있는 청취자를 자연스럽게 끌어들이기
2. 진입(4~7곡): 점점 깊어짐 — 속도와 볼륨 살짝 낮아짐
3. 깊은 이완(8~15곡): 가장 깊고 고요한 구간 — 수면 유도
4. 유지(16~18곡): 깊은 상태 유지 — 변화 최소화
5. 마무리(19~20곡): 극도로 잔잔 — 완전한 고요

## 피해야 할 것
- 갑작스러운 음량 변화
- 밝은 톤의 악기 (트럼펫, 일렉기타)
- 드럼/퍼커션
- 빠른 아르페지오

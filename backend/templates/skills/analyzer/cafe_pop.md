# 카페/POP 음악 분석 스킬

## 타겟 청취자 특성
- 카페 운영자 (매장 BGM)
- 재택근무/카페 작업자
- 드라이브/산책 중 감성 음악 청취자
- 20~40대 감성 취향

## 성공 채널 패턴
- 영상 길이: 1~3시간
- 곡 수: 15~25곡
- 썸네일: 따뜻한 톤, 커피/카페/창가 이미지
- 제목 키워드: "Cafe Music", "Acoustic", "Chill", "Morning", "Rainy Day"

## 분위기 흐름 최적 패턴
1. 시작(1~3곡): 밝고 경쾌하게 시작 — 활기찬 첫인상
2. 전개(4~8곡): 약간 차분하게 — 대화하기 좋은 볼륨
3. 중반(9~13곡): 다시 살짝 업템포 — 지루함 방지
4. 후반(14~17곡): 잔잔하고 감성적
5. 마무리(18~20곡): 따뜻한 마무리

## 피해야 할 것
- 과도하게 큰 비트
- 시끄러운 일렉기타 솔로
- 랩/힙합 요소
- 정치적/사회적 가사
